CSS3 --笔记 #

一、边框 #

1.border-radius 圆角 #

//border-radius:四个属性值，第一个值为左上角，第二个值为右上角，第三个值为右下角，第四个值为左下角。

css
#example1
{
	border:2px solid #a1a1a1;
	padding:10px 40px; 
	background:#dddddd;
	width:300px;
	border-radius:25px;
}
#example2 {
  border: 2px solid red;
  padding: 10px;
  border-radius: 50px 20px;
}

2.box-shadow 阴影 #

box-shadow : h-shadow v-shadow blur spread color

css
div{
	width:300px;
	height:100px;
	background-color:yellow;
	box-shadow: 10px 10px 10px #888888;
}

值说明
h-shadow必需的。水平阴影的位置。允许负值
v-shadow必需的。垂直阴影的位置。允许负值
blur可选。模糊距离
spread可选。阴影的大小
color可选。阴影的颜色。在CSS颜色值寻找颜色值的完整列表

3.border-image 边界图片 #

略

二、背景 #

1.background-image #

css
#example1 {
    background-image: url(img_flwr.gif), url(paper.gif);
    background-position: right bottom, left top;
    background-repeat: no-repeat, repeat;
    padding: 15px;
}

注：背景支持多个图片叠加

2.background-size #

指定背景图像的大小，可以指定像素或百分比大小，常用100%

css
div{
    background:url(img_flwr.gif);
    background-size:100% 100%;
    background-repeat:no-repeat;
}

3.background-origin图像位置 #

指定了背景图像的位置区域

css
div
{
	border:1px solid black;
	padding:35px;
	background-image:url('smiley.gif');
	background-repeat:no-repeat;
	background-position:left;
}
#div1
{
	background-origin:border-box;
}
#div2
{
	background-origin:content-box;
}

4.background-clip指定位置绘制 #

css
#example1 {
    border: 10px dotted black;
    padding:35px;
    background: yellow;
}

#example2 {
    border: 10px dotted black;
    padding:35px;
    background: yellow;
    background-clip: padding-box;
}

#example3 {
    border: 10px dotted black;
    padding:35px;
    background: yellow;
    background-clip: content-box;
}

三、渐变色 #

支持在两个或多个颜色之间平稳的过度

线性渐变：向下/向上/向左/向右/对角方向， 默认是向下

径向渐变：由它们的中心定义

1.linear-gradient线性渐变 #

不同方向的渐变

css
#grad1 {
    height: 200px;
		background-color: red; /* 浏览器不支持时显示 */
    background-image: linear-gradient(#e66465, #9198e5);
}

css
//向右
#grad {
  height: 200px;
  background-image: linear-gradient(to right, red , yellow);
}
//左上到右下
#grad {
  height: 200px;
  background-image: linear-gradient(to bottom right, red, yellow);
}

指定位置开始渐变

css
#grad3 {
    height: 200px;
	background-color: red; /* 浏览器不支持的时候显示 */
	background-image: linear-gradient(red 10%, green 80%, blue 100%); /* 标准的语法（必须放在最后），指定位置才开始变颜色 */
}

2.radial-gradient径向渐变 #

指定位置开始渐变

css
#grad1 {
    height: 150px;
    width: 200px;
    background-color: red; /* 浏览器不支持的时候显示 */
    background-image: radial-gradient(red 5%, green 15%, blue 60%); /* 标准的语法（必须放在最后） */
}

设置径向渐变的形状（默认ellipse，还有circle）

css
#grad1 {
    height: 150px;
    width: 200px;
    background-color: red; /* 浏览器不支持的时候显示 */
    background-image: radial-gradient(red, yellow, green); /* 标准的语法（必须放在最后） */
}

#grad2 {
    height: 150px;
    width: 200px;
    background-color: red; /* 浏览器不支持的时候显示 */
    background-image: radial-gradient(circle, red, yellow, green); /* 标准的语法（必须放在最后） */
}

四、文本效果 #

1.text-shadow文本阴影 #

text-shadow属性：水平阴影、垂直阴影、模糊距离和阴影的颜色

css
h1
{
    text-shadow: 5px 5px 5px #FF0000;
}

补充：伪元素 ::before ::after

  • 伪元素既然带一个“元素”，说明它和 p ，a ，h1 等一样，可以包裹住页面的一部分内容并对其中的内容施加一定的影响。我们可以通过样式表给其书写样式
  • 伪元素既然还带一个“伪”，说明它其实在 DOM 树中并不存在，只是逻辑上存在所以我们抽象出来的一个元素而已

2.text-overflow文本溢出 #

指定文本溢出属性如何显示

css
p.test1 {
    white-space: nowrap; 
    width: 200px; 
    border: 1px solid #000000;
    overflow: hidden;
    text-overflow: clip; 
}
 
p.test2 {
    white-space: nowrap; 
    width: 200px; 
    border: 1px solid #000000;
    overflow: hidden;
    text-overflow: ellipsis; 
}

3.word-break段落规定文本的换行规则 #

css
p.test1
{
	width:9em; 
	border:1px solid #000000;
	word-break:keep-all;
}

p.test2
{
	width:9em; 
	border:1px solid #000000;
	word-break:break-all;
}

五、2D转换 #

1.translate 平移 #

根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动。

css
div{
transform: translate(50px,100px);
-ms-transform: translate(50px,100px); /* IE 9 */
-webkit-transform: translate(50px,100px); /* Safari and Chrome */
}

2.rotate 旋转 #

在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转。

css
div{
transform: rotate(30deg);
-ms-transform: rotate(30deg); /* IE 9 */
-webkit-transform: rotate(30deg); /* Safari and Chrome */
}

3.scale 缩放 #

该元素增加或减少的大小,如下宽度变为2倍，高度变为3倍

css
div {
    margin: 150px;
    width: 200px;
    height: 100px;
    background-color: yellow;
    border: 1px solid black;
    border: 1px solid black;
    -ms-transform: scale(2,3); /* IE 9 */
    -webkit-transform: scale(2,3); /* Safari */
    transform: scale(2,3); /* 标准语法 */
}

4.skew 倾斜 #

skewX是指图形在X轴方向上（也就是水平方向上进行变形），千万不要理解为它是x轴进行旋转！！其次，图形的边一定是和对应方向上的坐标轴平行的！也就是说，竖直方向上的边一定是和y轴平行。

css
transform:skewX(30deg);

css
transform:skewY(10deg);

七、过渡 #

transition #

指定要添加效果的CSS属性

指定效果的持续时间

css
//如下例子，旋转180度过程中，长宽都会增加
div {
    width: 100px;
    height: 100px;
    background: red;
    -webkit-transition: width 2s, height 2s, -webkit-transform 2s; /* For Safari 3.1 to 6.0 */
    transition: width 2s, height 2s, transform 2s;
}
div:hover {
    width: 200px;
    height: 200px;
    -webkit-transform: rotate(180deg); /* Chrome, Safari, Opera */
    transform: rotate(180deg);
}

八、图片 #

1.圆角图片 #

border-radius 单位可以是px也可以是百分比

css
img {
    border-radius: 8px;
}

2.响应式图片 #

响应式图片会自动适配各种尺寸的屏幕，如下当浏览器窗口大于图片宽度时，图片宽度仍旧原来大小

css
img {
    max-width: 100%;
    height: auto;
}

3.响应式图片相 #

根据浏览器尺寸来设置不同的显示样式，如为了适配不同客户端类型

css
.responsive {
    padding: 0 6px;
    float: left;
    width: 24.99999%;
}

@media only screen and (max-width: 700px){
    .responsive {
        width: 49.99999%;
        margin: 6px 0;
    }
}

@media only screen and (max-width: 500px){
    .responsive {
        width: 100%;
    }
}

4、图片Modal #

实现点击图片弹出图片

html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8"> 
<title>菜鸟教程(runoob.com)</title> 
<style>
#myImg {
    border-radius: 5px;
    cursor: pointer;
    transition: 5.3s;
}

#myImg:hover {opacity: 0.7;}

/* The Modal (background) */
.modal {
    display: block; /* Hidden by default */
    position: fixed; /* Stay in place */
    z-index: 1; /* Sit on top */
    padding-top: 100px; /* Location of the box */
    left: 0;
    top: 0;
    width: 100%; /* Full width */
    height: 100%; /* Full height */
    overflow: auto; /* Enable scroll if needed */
    background-color: rgb(0,0,0); /* Fallback color */
    background-color: rgba(0,0,0,0.9); /* Black w/ opacity */
}

/* Modal Content (image) */
.modal-content {
    margin: auto;
    display: block;
    width: 80%;
    max-width: 700px;
}

/* Caption of Modal Image */
#caption {
    margin: auto;
    display: block;
    width: 80%;
    max-width: 700px;
    text-align: center;
    color: #ccc;
    padding: 10px 0;
    height: 150px;
}

/* Add Animation */
.modal-content, #caption {    
    -webkit-animation-name: zoom;
    -webkit-animation-duration: 0.6s;
    animation-name: zoom;
    animation-duration: 0.6s;
}

@-webkit-keyframes zoom {
    from {-webkit-transform: scale(0)} 
    to {-webkit-transform: scale(1)}
}

@keyframes zoom {
    from {transform: scale(0.1)} 
    to {transform: scale(1)}
}

/* The Close Button */
.close {
    position: absolute;
    top: 15px;
    right: 35px;
    color: #f1f1f1;
    font-size: 40px;
    font-weight: bold;
    transition: 0.3s;
}

.close:hover,
.close:focus {
    color: #bbb;
    text-decoration: none;
    cursor: pointer;
}

/* 100% Image Width on Smaller Screens */
@media only screen and (max-width: 700px){
    .modal-content {
        width: 100%;
    }
}
</style>
</head>
<body>

<h2>图片模态框</h2>
<p>本实例演示了如何结合 CSS 和 JavaScript 来一起渲染图片。</p><p>
首先，我们使用 CSS 来创建 modal 窗口 (对话框), 默认是隐藏的。<p>
<p>然后，我们使用 JavaScript 来显示模态窗口，当我们点击图片时，图片会在弹出的窗口中显示：</p>
<img id="myImg" src="http://www.runoob.com/wp-content/uploads/2016/04/img_lights.jpg" alt="Northern Lights, Norway" width="300" height="200">

<!-- The Modal -->
<div id="myModal" class="modal">
  <span class="close">×</span>
  <img class="modal-content" id="img01">
  <div id="caption"></div>
</div>

<script>
// 获取模态窗口
var modal = document.getElementById('myModal');

// 获取图片模态框，alt 属性作为图片弹出中文本描述
var img = document.getElementById('myImg');
var modalImg = document.getElementById("img01");
var captionText = document.getElementById("caption");
img.onclick = function(){
    modal.style.display = "block";
    modalImg.src = this.src;
    modalImg.alt = this.alt;
    captionText.innerHTML = this.alt;
}

// 获取 <span> 元素，设置关闭模态框按钮
var span = document.getElementsByClassName("close")[0];

// 点击 <span> 元素上的 (x), 关闭模态框
span.onclick = function() { 
    modal.style.display = "none";
}
</script>

</body>
</html>

九、多媒体查询 #

CSS2媒体类型可用于检查（显示器、便携设备、电视机等不同终端进而设置不同的样式规则）

CSS3多媒体可用于检测很多事情：viewport的宽度和高度、设备的宽度高度、朝向（智能手机横屏、竖屏）、分辨率

css
@media not|only mediatype and (expressions) {
    CSS 代码...;
}
css
//当浏览器的宽度在 700 到 1000px, 会在邮箱链接前添加 "Email: ":
@media screen and (max-width: 1000px) and (min-width: 700px) {
    ul li a:before {
        content: "Email: ";
        font-style: italic;
        color: #666666;
    }
}
