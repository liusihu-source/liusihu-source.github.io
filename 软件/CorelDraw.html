CorelDraw --笔记 #

一、界面 #

  • 菜单栏
  • 功能栏
  • 工具栏
  • 工具组
    • 不同工具的属性栏
  • 文件标题栏
    • 一个文件中可有多页
  • 工作区
  • 状态栏

二、基础操作 #

界面操作 #

  • 放大缩小
    • shift按键可以缩小
    • 滚轮键向前放大
    • 滚轮键向后缩小
  • 平移
    • 按住滚轮键
  • 颜色
    • 填充颜色左键填充，右键是边框颜色更改

对象选择 #

  • 取消所有对象组合

  • 手绘选择

    • alt键再选中对象的一点就可选中全部

  • 形状工具

    • 锚点更改位置

其他操作 #

  • 复制 ctrl C/V
  • 填充
    • 鼠标左键右侧的调色板
    • 鼠标右键右侧的调色板
      • 下方状态栏有更为详尽的颜色选择方案

三、文本工具 #

  • 工具组中的字
    • 直接单击：美术字
      • 工具属性栏
        • XY : 所在位置
        • 放大缩小字体
      • 单击中间十字架——切换旋转和放大缩小
      • 水平 / 竖直 翻转
      • 美术字 和 段落文本的对齐方式
      • 文本属性
        • 内部有多项属性功能
      • 其他：文本转曲，可以实现矢栅之间转换
    • 拉选框：段落文本
      • 工具属性栏
      • 文本属性
  • 工具组中地路径文本
    • 画好路径后--文本选中对应周边轮廓即可

四、制表符 #

  • 可单行 单列 多行 多列更改大小
  • 制表符可通过调整水平、垂直来设置居中
  • 制表符的外边框、内边框或者单一行等可调整边框粗细
  • 通过顶部菜单栏 --表格，可将文字转换为表格

五、绘图工具 #

  • 手绘工具可调整线宽、线形状等
  • 两点线可绘制对应切线、垂直线
  • 三点曲线、贝塞尔曲线等
  • 钢笔工具，按住后移动可调整曲度 + 形状工具，可以调整节点所在位置及弯曲度

补：转换为曲线后就可以对节点进行更改设置

六、矩形工具 #

  • 随便画出矩形
  • 如果需要正方形，需要按住Ctrl
  • 椭圆，如果需要画标准圆，按住Ctrl
  • 其他形状工具

七、裁剪工具组 #

  • 裁剪工具，可裁剪出需要的图片区域。

  • 刻刀工具，可将图片裁剪成两部分，分别用

  • 虚拟段删除，可以删除内部的连接线

    • 内部删除完毕后，可以将外部轮廓线通过顶部的焊接工具将边框焊接为一个整体

  • 橡皮擦工具，可以对转曲后的图形擦除工作

八、对象编辑 #

  • 泊坞窗
    • 变换：位置、旋转、缩放和镜像、大小和倾斜
      • 注意副本的数目是否为0
    • 对齐与分布
      • 用于调整对象之间的对齐方式和中间的间隔
  • 对象造型工具（同ArcMap空间分析工具箱）
    • 焊接
    • 修建
    • 相交
    • 移除前方/后方对象
    • 创建边界

九、阴影工具 #

  • 阴影：可调整阴影的方向、程度以及羽化程度（模糊程度）
  • 轮廓图
    • 向内或向外增加轮廓
    • 更改轮廓数目、各轮廓间的宽度、顶角样式等
  • 混合
    • 显示出两个图形之间的相互转化过程
  • 变形和封套，可作出很多不规则图形，如旋转手里剑
  • 立体化
    • 可将某一几何对象改为立体图形
  • 块阴影
    • 可朝某个方向拉出一片阴影区域

十、度量工具 #

  • 水平、垂直或倾斜度量
  • 线段度量
  • 其他标注
  • 锚点工具
