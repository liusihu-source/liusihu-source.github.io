2.微任务 & 宏任务 #

javascript
console.log(1);
let a = setTimeout( () => console.log(2) )
console.log(3)
Promise.resolve(4).then(b => {
	console.log(b)
	clearTimeout(a)
})
console.log(5)

1354

  • 先执行同步135，定时器为宏任务，Promise.then()为微任务，因此先执行微任务，微任务中清除了定时器，所以不会有2，其中宏任务和微任务均属于异步
    • 宏任务：消息队列中等待被主线程执行的事件，宏任务执行会重新创建栈，宏任务执行结束后，栈也会随之销毁【】
    • 微任务：需要被异步执行的函数。【】 总的执行顺序：宏任务中的同步任务 -> 微任务当中的同步任务 ->微任务中的异步任务 -> 宏任务中的异步任务

1.数组length #

  • 数组length属性总比数组最后一个元素的下标大1
  • 对于后续更改length属性
    • 如果新值 < 当前值，数组将被截断
    • 如果新值 > 当前值，数组将被扩大，扩大的元素值为underfined

2. let作用域 #

javascript
for(let i=0;i<2;i++){
    setTimeout(function(){
    console.log(i)
    },100);
}
for(var i=0;i<2;i++){
    setTimeout(function(){
        console.log(i)
    },100);
}

0122

  • let 有自己的作用域块，在for循环中满足每次i输出一次
  • var没有自己的作用域，又因为settimeout是异步的，所以当异步任务开始前，i已经是2了

2.共享原型链，堆地址相同 #

javascript
function Father(age){
    this.age = age
}
function Son(age){
    Father.call(this);
}
Son.prototype = Father.prototype;
Father.prototype.getAge = function(){console.log(40);}
Son.prototype.getAge = function(){console.log(18);}
var father = new Father(40);
var son = new Son(18);
son.getAge();
father.getAge();

18 18 因为有两个原型链的地址相同，所以以最终赋值结果输出。

2. || && 运算符 #

  • ||
    • 如果第一个结果是true，则返回该操作数的值；如果false则一直迭代，直到存在结果为真，否则返回最后一个元素
    • 如 let flag1 = null || underfined，因为都是false，所以返回最后的undefined
  • &&
    • 如果有一个值为false，则返回该值；当所有值都为true，则返回最后一个操作数
    • 如let flag2 = null && undefined，因为第一个就是false，所以返回第一个null

1.闭包、函数和变量提升练习题 #

javascript
var a = 10; 
(function a() {
    a = 20;
    console.log(a); 
})()

会输出函数a的内容

  • 首先是变量提升，a = undefined
  • 之后由于函数提升优先级高于变量赋值，所以先a被赋值函数
  • 闭包函数，所以内部即刻输出
    • 其中 a = 20，意思是将整个函数赋值为20，并不是新的变量a
  • 输出a的函数体后，最后才是全局a再次赋值为10

3.变量回收机制 #

javascript
var i = 1;
var i = 2;
var add = function() {
	var i = 0;
	return function(){
		i++;
		console.log(i)
	}
}();
add();

上述代码执行后，有 3 个变量没有被回收

  • 首先全局变量 i 没有回收，其中第二行的var 没有作用，只是做了更新赋值
  • 第二个函数add变量
  • add函数内部变量i，因为该闭包中的局部变量倍 return的结果作用域引用，所以没有被回收

1.parseFloat() & toFixed() & Math.round & toPrecision() #

  • parseFloat() 返回一个浮点数
    • parseFloat(0.2 + 0.4) //0.600000000001
  • toFixed() 把数字转换为字符串，结果的小数点后有指定位数的数字
    • parseFloat((0.2 + 0.4).toFixed(1)) //'0.6'
  • Math.round 返回四舍五入的整数
    • Math.round(0.2 + 0.4) //1
  • toPrecision 把数字格式化为指定的长度
    • parseFloat((0.2 + 0.6).toPrecision(1)) //0.8
